# Morrigan Releases

Official releases of Morrigan client applications.

## Overview

This repository hosts stable releases of the Morrigan client applications. Download the latest version for your platform from the [Releases page](../../releases).

## Downloads

Visit the [Releases page](../../releases) to download the latest version for your operating system:

- **Windows**: Download the `.exe` installer or `.zip` archive
- **macOS**: Download the `.dmg` or `.pkg` installer  
- **Linux**: Download the `.deb`, `.rpm`, or `.AppImage` package

## Installation

### Windows
1. Download the Windows installer (`.exe`)
2. Run the installer and follow the setup wizard
3. Launch Morrigan from the Start Menu

### macOS
1. Download the macOS installer (`.dmg`)
2. Open the `.dmg` file
3. Drag Morrigan to your Applications folder
4. Launch from Applications

### Linux
1. Download the appropriate package for your distribution
2. Install using your package manager:
   - **Debian/Ubuntu**: `sudo dpkg -i morrigan-client_*.deb`
   - **Fedora/RHEL**: `sudo rpm -i morrigan-client-*.rpm`
   - **AppImage**: Make executable with `chmod +x` and run directly
