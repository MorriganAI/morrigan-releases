# MorriganAI Client

Official releases of the MorriganAI Desktop Client for Windows.

## Download

**[Download Latest Version](https://github.com/MorriganAI/morrigan-releases/releases/latest/download/MorriganClientInstaller.exe)**

## Installation

1. Download the installer using the link above
2. Run `MorriganClientInstaller.exe`
3. Follow the installation wizard
4. Launch MorriganAI Client from the Start Menu
5. Sign in with your MorriganAI account

## System Requirements

- Windows 10 or later (64-bit)
- 100 MB free disk space
- Internet connection
